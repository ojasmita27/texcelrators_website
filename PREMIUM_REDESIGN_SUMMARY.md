## 🎨 Design System Philosophy

### Color Palette
```
Dark Background:     #0a0e27 → #050914
Cards:              #1a1f3a → #121c34
Primary Accent:     #0099ff (Blue)
Secondary Accent:   #00d4ff (Cyan)
Text Primary:       #ffffff
Text Secondary:     #b0b5c1
Success:            #43e97b
Warning:            #ffc43d
Danger:             #ff6b6b
```

### Spacing Scale
```
Compact:    8px
Normal:     12-14px
Medium:     18-20px
Large:      24-28px
Extra:      32-40px
```

### Typography Scale
```
Labels:     0.78-0.88rem (uppercase)
Body:       0.95-1rem
Headings:   1.1-1.2rem
Values:     1.6-2rem
Large:      1.32rem+
```

### Shadow System
```
Subtle:     0 8px 20px rgba(0,0,0,0.2)
Medium:     0 12px 32px rgba(0,0,0,0.32)
Heavy:      0 20px 52px rgba(0,0,0,0.48)
Glow:       0 0 24-60px rgba(0,153,255,0.05-0.12)
```

### Animation Speeds
```
Quick:      280ms (interactions)
Normal:     320ms (hover effects)
Smooth:     420ms (transitions)
Easing:     cubic-bezier(0.2, 0.9, 0.2, 1)
```
